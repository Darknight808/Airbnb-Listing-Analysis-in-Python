# Airbnb Listings Data Analysis

> Data analysis project exploring Airbnb listings to uncover pricing trends, location patterns, and key factors affecting the rental market using Python.

---

## 📝 Project Overview
This project analyzes Airbnb listing data to answer key questions such as:
- What factors influence listing prices?
- How do prices vary by location and property type?
- What patterns exist in availability and demand?

The project simulates a real-world business scenario in the **hospitality and short-term rental market**.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- GitHub

---

## 📊 Dataset
The dataset contains Airbnb listings with features such as:
- Price
- Neighborhood / Location
- Room type
- Availability
- Number of reviews

*(Dataset source provided in the project or video)*

---

## 🔍 Key Analysis & Insights
- **Location impact:** Prices vary significantly depending on neighborhood  
- **Room type:** Entire homes/apartments are more expensive than private/shared rooms  
- **Demand patterns:** Listings with more reviews tend to have higher occupancy  
- **Availability:** Some areas show higher availability, indicating lower demand or oversupply
